# spring-boot-boilerplate
Launch a spring boot based project quickly with basic features 

## features

- actuator
- aop
- cache
- JPA
- HATEOAS
- mail
- webflux
- flyway
- session
- lombok
- JMS


## run 

- start mysql & redis `docker-compose --file docker/docker-compose.yml up`
- run spring boot application `./gradlew bootRun`
